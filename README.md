# 🎓 Sistema de Gerenciamento Escolar (Lógica em Python)

Este repositório contém um script em **Python** desenvolvido para praticar conceitos fundamentais de lógica de programação, manipulação de estruturas de dados (listas e dicionários) e controle de fluxo.

## 📝 Descrição do Projeto

O sistema simula o controle de notas de uma instituição de ensino, permitindo o cadastro de alunos, cálculo automático de médias e geração de relatórios estatísticos da turma.

## 🚀 Funcionalidades

-   **📥 Cadastro de Alunos:** Registro de nome e duas notas.
-   **📊 Cálculo de Média:** Processamento automático da média aritmética.
-   **⚖️ Verificação de Situação:** -   **Aprovado:** Média ≥ 7.0
    -   **Recuperação:** Média entre 5.0 e 6.9
    -   **Reprovado:** Média < 5.0
-   **📋 Relatório Completo:** Exibição de todos os alunos cadastrados.
-   **📈 Estatísticas da Turma:** -   Média geral da turma.
    -   Destaque para a maior e menor média.
    -   Contagem total de alunos.

## 🛠️ Tecnologias Utilizadas

* [Python 3.x](https://www.python.org/)
* Conceitos de Dicionários e Listas
* Estruturas de Repetição (`while`) e Condicionais (`if/elif/else`)
